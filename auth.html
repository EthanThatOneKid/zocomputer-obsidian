Zo Computer: Obsidian: Lock It Down

The server supports three access modes. The prompt below sets full mode so only authenticated users can read or write. 

Add authentication to Obsidian Web.

Stop the server if it is running.
Set these environment variables:
  AUTH_MODE=full
  OBSIDIAN_WEB_USERNAME=ethan
  OBSIDIAN_WEB_PASSWORD=your-secure-password
  OBSIDIAN_WEB_REALM=Obsidian Web

Then restart the server. After starting, make a request to
http://127.0.0.1:3000 and tell me what HTTP status code it returns.
A 302 redirect means the login page is protecting the app.

The table below shows what each mode does. If you do not set a password, auth stays off no matter which mode you choose.

ModeReadWriteUse Case
fullAuth onlyAuth onlyPrivate vault
readonlyAnyoneAuth onlyPublic read
disabledAnyoneAnyoneFully open
