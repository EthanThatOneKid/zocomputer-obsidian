Zo Computer: Obsidian: Set It Up

Your notes stay as normal Markdown files. The browser is just the window into your vault. You run the Obsidian web renderer on your own Zo.

This is not remote desktop. No Electron needed. obsidian-web loads Obsidian's original renderer unmodified and replaces every Electron dependency with lightweight HTTP shims.

The prompt below sets up the server from scratch. Copy it and paste it to your Zo agent. 

Set up Obsidian Web on my Zo so I can access my Obsidian vault from a browser.

1. Clone https://github.com/EthanThatOneKid/obsidian-web.git into my home directory and name it obsidian-web.

2. Inside obsidian-web, create the vendor directory and run node scripts/update-obsidian.js to download the Obsidian browser renderer.

3. Install the server dependencies by running npm install inside the src/server directory.

4. Start the server on port 3000 with AUTH_MODE set to disabled. Do not add any authentication yet.

After each step, confirm it succeeded before moving on. When the server is running, tell me the HTTP status code from http://127.0.0.1:3000.

Once the server is running, choose how to configure it.
