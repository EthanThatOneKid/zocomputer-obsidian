# Video production checklist

## Before recording

- [ ] Test-run both prompts with Zo so you know expected output and timing for each
- [ ] Have the obsidian-web login page pre-loaded from a test run (Tab 2)
- [ ] Confirm the obsidian-web instance loads in a browser on a separate machine (or incognito)
- [ ] Set up lighting for face cam — consistent throughout (single take, no re-lighting)
- [ ] Close unnecessary windows and tabs on the recording machine
- [ ] Disable notifications (macOS: Do Not Disturb / Windows: Focus Assist)
- [ ] Arrange three windows so alt-tab cycles in order: Tab 1 (setup page) → Zo → Tab 2 (obsidian-web)

## Recording session

- [ ] Tab 1 — setup page is open, scrolled to the password-protected prompt, copy button visible
- [ ] Zo window — fresh session, cursor in the input field
- [ ] Tab 2 — obsidian-web login page, ready to log in
- [ ] Press record. One take, no cuts.
- [ ] Face cam intro → switch to Tab 1 → scroll both prompts → copy → switch to Zo → paste → narrate over execution → result shown → switch to Tab 2 → log in → show vault → face cam outro
- [ ] Hold on the end card for 3 seconds before stopping

## Post-production

- [ ] Add end card (static, 3s hold) if not recorded in the take
- [ ] Add soft sound effect on the copy-click (optional)
- [ ] Export at 1920x1080, H.264, ~15-20 Mbps

## Publishing

- [ ] Upload to YouTube (public or unlisted)
- [ ] Update README with video link
- [ ] Consider a Twitter/X clip from the Zo agent demo section (~20s)
