# Video production checklist

## Before recording

- [ ] Test Zo agent with the setup prompt so you know execution time and expected output
- [ ] Confirm the `obsidian-web` instance loads in a browser on a separate machine (or incognito)
- [ ] Set up lighting for face cam — same position for intro and outro
- [ ] Close unnecessary windows/tabs on recording machine
- [ ] Disable notifications (macOS: Do Not Disturb / Windows: Focus Assist)

## Recording session

- [ ] Shot 1 — Intro face cam (0:00-0:18). Record a few takes, pick the best.
- [ ] Shot 2 — Landing page + copy prompt. One clean take. Cursor slow and deliberate.
- [ ] Shot 3 — Zo agent window. Paste prompt, let it run. Capture the full stream.
- [ ] Shot 4 — Obsidian Web instance loading the vault. First load from cold.
- [ ] Shot 5 — Private page + public page scroll. One continuous scroll through both.
- [ ] Shot 6 — Outro face cam (1:18-1:40). Record after other shots so energy matches.
- [ ] Optional — Split-screen still for editing reference.

## In edit

- [ ] Speed up Shot 3 (Zo agent response) to ~1.5x if execution exceeds 30s on screen
- [ ] Add soft sound effect on the copy-click (Shot 2)
- [ ] Face cam opacity: full in intro/outro. Cut away cleanly during demo.
- [ ] No music track, or very quiet ambient drone in demo section only
- [ ] Add end card (static, 3s hold)
- [ ] Export at 1920x1080, H.264, ~15-20 Mbps

## Post-production

- [ ] Upload to YouTube (public or unlisted)
- [ ] Update README with video link
- [ ] Consider a Twitter/X clip from the Zo agent demo section (~20s)
