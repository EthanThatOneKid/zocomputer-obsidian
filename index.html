Zo Computer: Obsidian

Obsidian is a knowledge base that works on local Markdown files. Zo lets you run the Obsidian web renderer on your own server so your notes stay yours and you can reach them from any browser.

Zo Computer: Obsidian is a prompt reference for setting up obsidian-web on Zo. Each page has a prompt you can copy and paste to your Zo agent. Zo understands the prompt and sets everything up for you.

  • Hello, Obsidian
  • Clone the Repo
  • Download the Renderer
  • Install Dependencies
  • Start the Server
  • Point at Your Vault
  • Lock It Down
