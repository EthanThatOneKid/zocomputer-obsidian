Zo Computer: Obsidian

Obsidian is a knowledge base that works on local Markdown files. Zo lets you run the Obsidian web renderer on your own server so your notes stay yours and you can reach them from any browser.

Zo Computer: Obsidian is a prompt reference for setting up obsidian-web on Zo. Pick the mode you want, copy the prompt, and paste it to your Zo agent. Zo understands the prompt and sets everything up for you.

  • Set It Up — Choose: password-protected (default) or public readonly. One prompt clones, installs, configures auth, and starts the server.
