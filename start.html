Zo Computer: Obsidian: Start the Server

Start the Node.js server from the src/server directory. By default it binds to 127.0.0.1:3000. 

$ cd src/server
$ npm start

For development with auto-reload on file changes: 

$ npm run dev

Once running, verify the server responds: 

$ curl -s -o /dev/null -w "%{http_code}" http://127.0.0.1:3000/
200

Open http://127.0.0.1:3000 in a browser. You should see the Obsidian loading screen with its dark theme and spinner. 

$ curl -s http://127.0.0.1:3000/ | head -5
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8"/>
  <title>Obsidian Web</title>
