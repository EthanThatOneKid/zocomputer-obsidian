Zo Computer: Obsidian: Start the Server

The server binds to 127.0.0.1:3000 by default. Give Zo this prompt to start it without authentication so you can test. 

Start the Obsidian Web server.

Go to the src/server directory inside obsidian-web.
Start the server by running: AUTH_MODE=disabled node index.js
The server should bind to 127.0.0.1 on port 3000.

After starting, verify it responds by making a request to http://127.0.0.1:3000.
