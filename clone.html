Zo Computer: Obsidian: Clone the Repo

Start by cloning the obsidian-web repository. This gives you the server code, shims, and the project structure. 

$ git clone https://github.com/EthanThatOneKid/obsidian-web.git
$ cd obsidian-web

The repo contains the Node.js server, client runtime, Cloudflare deployment option, and the scripts that fetch Obsidian's renderer. 

$ ls -F
scripts/  src/  user-data/  .gitignore  README.md
