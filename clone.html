Zo Computer: Obsidian: Clone the Repo

The obsidian-web repository contains the server code, shims, and the project structure. Give Zo this prompt to clone it. 

Clone https://github.com/EthanThatOneKid/obsidian-web.git into my home directory.
Name the folder obsidian-web.
After cloning, confirm the directory exists and contains scripts, src, and user-data folders.

Zo should confirm the cloned directory contents. The repo ships with a demo vault in user-data/demo-vault so you can test before connecting your own vault. 

List the contents of the obsidian-web directory.
Show me what folders and files are at the top level.
