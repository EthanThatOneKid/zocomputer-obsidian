Zo Computer: Obsidian: Install Dependencies

Navigate to the server directory and install the Node.js dependencies. 

$ cd src/server
$ npm install

The server depends on Express for HTTP, ws for WebSocket sync, chokidar for file watching, and compression for gzip responses.
