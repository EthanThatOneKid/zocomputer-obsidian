Zo Computer: Obsidian: Point at Your Vault

Set VAULT_PATH to point at your notes folder. The default is user-data/demo-vault if you don't set it. 

$ export VAULT_PATH=/path/to/your/vault
$ npm start

The bootstrap API confirms the vault contents loaded. Here's what the demo vault looks like: 

$ curl -s http://127.0.0.1:3000/api/bootstrap | jq '.dirs[""]'
[
  { "name": "Linked notes.md", "isFile": true, "size": 74 },
  { "name": "Welcome.md", "isFile": true, "size": 300 },
  { "name": "folder", "isDirectory": true },
  { "name": "ללא כותרת.base", "isFile": true, "size": 42 }
]

Your vault stays a folder of plain files on disk. The browser is only the window into that folder.

If you need to change the vault later without restarting, visit /starter and enter an absolute path.
