Zo Computer: Obsidian: Point at Your Vault

You can serve any folder of Markdown files as the vault. Replace the path below with the absolute path to your notes directory. The default vault is user-data/demo-vault if you want to test first. 

Point Obsidian Web at my vault.

Stop the server if it is running.
Set the VAULT_PATH environment variable to the absolute path of my Obsidian vault.
Then restart the server with AUTH_MODE=disabled.

After starting, check the bootstrap API at http://127.0.0.1:3000/api/bootstrap
and show me the file listing that was loaded.
