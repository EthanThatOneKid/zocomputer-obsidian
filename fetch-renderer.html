Zo Computer: Obsidian: Download the Renderer

Obsidian's browser renderer files are not bundled in the Git repo. Zo needs to download them from the official release using the setup script. 

Download the Obsidian renderer for my obsidian-web setup.

Inside the obsidian-web directory, run node scripts/update-obsidian.js.
If the vendor directory does not exist, create it first before running the script.
The script fetches the latest release, verifies its checksum, and extracts
about 352 files into vendor/obsidian.
