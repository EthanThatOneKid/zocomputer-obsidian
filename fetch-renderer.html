Zo Computer: Obsidian: Download the Renderer

Run the setup script to download Obsidian's browser-facing renderer files from the official release. 

$ node scripts/update-obsidian.js

The script fetches the .asar.gz release asset, verifies its SHA-256 digest, and extracts the renderer into vendor/obsidian/. 

Obsidian release: 1.12.7 (v1.12.7)
Selected asset: obsidian-1.12.7.asar.gz
Downloading obsidian-1.12.7.asar.gz
Verified SHA-256: 75dd34f14c9db558fbad19e80f0b201bc9805b51b7388370277e0f91a38bd850
Decompressing obsidian-1.12.7.asar.gz
Extracting obsidian-1.12.7.asar
Extracted 352 files (23968823 bytes)
Updated vendor/obsidian to Obsidian 1.12.7

You can pin a specific version if needed: 

$ node scripts/update-obsidian.js --version 1.12.7
